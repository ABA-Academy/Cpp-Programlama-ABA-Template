# Soru 4: 1'den N'e Kadar Sayılar

## Zorluk: ⭐⭐ Orta

## Açıklama

Kullanıcıdan bir pozitif tam sayı N alın ve 1'den N'e kadar olan tüm sayıları ekrana yazdırın.

## İstenenler

1. Kullanıcıdan pozitif bir tam sayı N alın
2. 1'den N'e kadar (N dahil) tüm sayıları yazdırın
3. Her sayı ayrı satırda olmalı

## Örnekler

### Örnek 1
**Girdi:** `5`
**Çıktı:**
```
1
2
3
4
5
```

### Örnek 2
**Girdi:** `3`
**Çıktı:**
```
1
2
3
```

### Örnek 3
**Girdi:** `1`
**Çıktı:**
```
1
```

## İpuçları

- `for` döngüsü kullanın: `for(int i = 1; i <= n; i++)`
- Her sayıyı `cout << i << endl;` ile yazdırın
