# Soru 1: Hello World

## Zorluk: ⭐ Başlangıç

## Açıklama

İlk C++ programınızı yazın! Ekrana "Hello, World!" yazdıran bir program oluşturun.

## İstenenler

Programınız çalıştırıldığında şu çıktıyı vermelidir:
```
Hello, World!
```

## Örnekler

### Çıktı
```
Hello, World!
```

## İpuçları

- `cout` kullanarak ekrana yazı yazdırabilirsiniz
- Satır sonu için `endl` veya `\n` kullanabilirsiniz
- `using namespace std;` satırını unutmayın

## Nasıl Test Ederim?

1. Kodunuzu `main.cpp` dosyasına yazın
2. Terminalde `gonder` komutunu çalıştırın
3. Otomatik testler çalışacak ve sonucu göreceksiniz
