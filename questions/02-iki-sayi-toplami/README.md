# Soru 2: İki Sayının Toplamı

## Zorluk: ⭐ Başlangıç

## Açıklama

Kullanıcıdan iki tam sayı alın ve toplamını ekrana yazdırın.

## İstenenler

1. Kullanıcıdan iki tam sayı alın
2. Bu sayıların toplamını hesaplayın
3. Sonucu ekrana yazdırın

## Örnekler

### Örnek 1
**Girdi:**
```
5
3
```
**Çıktı:**
```
8
```

### Örnek 2
**Girdi:**
```
-10
7
```
**Çıktı:**
```
-3
```

### Örnek 3
**Girdi:**
```
100
200
```
**Çıktı:**
```
300
```

## İpuçları

- `cin` kullanarak kullanıcıdan girdi alabilirsiniz
- İki `int` değişken tanımlayın
- `+` operatörü ile toplayın
- Sonucu `cout` ile yazdırın

## Nasıl Test Ederim?

1. Kodunuzu `main.cpp` dosyasına yazın
2. Terminalde `gonder` komutunu çalıştırın
3. Otomatik testler farklı sayılarla programınızı test edecek
